Verdict

Which plugin for which use case?

Three very different tools for three very different contexts.

EfficientIP official
Legacy environments only
  • SOAP API — not compatible with vRO 8.x
  • No DNS, no alias management
  • No form actions for vRA
  • Unmaintained since ~2021
Only viable if you are still running vRO 7.x and have no intention of upgrading. Not suitable for production automation at scale.
JuliaLblnd Python plugin
Narrow vRA IPAM use case
  • Native vRA 8 IPAM Provider integration
  • REST API, open source MIT
  • Requires Docker infrastructure
  • No DNS, no aliases, no workflows
  • Last commit October 2023
Good fit if you only need automatic IP assignment during VM lifecycle in vRA 8, and are comfortable maintaining a Python/Docker stack yourself.
AUTOMACLOUD plugin
Full-stack production solution
  • Full DNS + IPAM + alias lifecycle
  • Idempotent — safe to retry
  • 7 vRA form actions for dynamic dropdowns
  • Ghost machine detection (ping check)
  • Offline license — air-gap ready
  • Actively maintained in 2026
  • No extra infrastructure required
The only option covering the complete DNS/IPAM automation lifecycle on vRO 8.x.
